# 用例规约表草案

下面是可以直接放进论文第2章的用例规约表草案。

## 表1 查看模型评估结果用例规约表

| 项目 | 内容 |
|---|---|
| 用例编号 | UC-01 |
| 用例名称 | 查看模型评估结果 |
| 功能描述 | 用户进入模型评估页面，查看不同模型的 MAE、RMSE、MAPE 指标结果 |
| 执行者 | 系统用户 |
| 前置条件 | 系统已正常启动，后端接口可访问，模型评估结果文件存在 |
| 后置条件 | 模型评估结果被成功展示在页面中 |
| 涉众利益 | 帮助用户快速比较各模型预测性能，为选择默认模型提供依据 |
| 基本路径 | 1. 用户进入模型评估页面；2. 前端调用模型评估接口；3. 后端读取评估结果文件；4. 系统返回模型指标数据；5. 页面展示结果表 |
| 扩展路径 | 当接口异常或文件不存在时，系统提示加载失败 |
| 字段列表 | 模型名称、MAE、RMSE、MAPE |
| 业务规则 | 默认按照 RMSE 从低到高排序，高亮推荐模型 |
| 备注 | 可与图表中心联动展示指标图 |

## 表2 查看未来预测结果用例规约表

| 项目 | 内容 |
|---|---|
| 用例编号 | UC-02 |
| 用例名称 | 查看未来预测结果 |
| 功能描述 | 用户在预测结果页面中查看未来12个月的不同模型预测值 |
| 执行者 | 系统用户 |
| 前置条件 | 预测结果文件已生成，前端页面可正常访问 |
| 后置条件 | 预测结果表成功展示 |
| 涉众利益 | 便于用户查看未来销量变化趋势，为库存与销售决策提供支持 |
| 基本路径 | 1. 用户进入预测结果页面；2. 系统请求预测结果接口；3. 后端返回预测表；4. 页面展示预测数据 |
| 扩展路径 | 若预测文件缺失，则提示结果不可用 |
| 字段列表 | 月份、随机森林、梯度提升回归、极端随机森林、LSTM、线性回归 |
| 业务规则 | 用户可以按月份检索，也可以切换重点模型进行查看 |
| 备注 | 可与模型详情接口联动 |

## 表3 查看指定模型预测图用例规约表

| 项目 | 内容 |
|---|---|
| 用例编号 | UC-03 |
| 用例名称 | 查看指定模型预测图 |
| 功能描述 | 用户在图表中心选择某个模型，系统展示对应的 Pyecharts 预测曲线 |
| 执行者 | 系统用户 |
| 前置条件 | 图表接口可正常访问，指定模型存在 |
| 后置条件 | 系统展示该模型的预测图表 |
| 涉众利益 | 帮助用户直观对比不同模型的趋势变化，提升分析效率 |
| 基本路径 | 1. 用户进入图表中心；2. 用户选择模型；3. 前端请求模型图表接口；4. 后端生成 Pyecharts 图；5. 页面嵌入显示图表 |
| 扩展路径 | 若模型名称非法，则系统提示模型不存在 |
| 字段列表 | 模型名称、月份、预测值 |
| 业务规则 | 模型切换后图表应立即刷新 |
| 备注 | 图表与预测结果表保持一致的数据来源 |

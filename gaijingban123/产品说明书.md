# 牙膏销量预测系统 · 产品说明书

## 1. 系统简介

本系统是一套面向牙膏销售业务的月度销量预测平台，支持历史样本维护、多模型训练与评估、未来销量预测展示、AI 智能分析和交互式图表中心。系统采用离线训练 + 在线展示的架构，适合定期更新数据后重新训练、即时查看预测结果的业务场景。

---

## 2. 技术架构

| 层级       | 技术选型                        |
|------------|-------------------------------|
| 前端       | Vue 3 + Vite                  |
| 后端       | Django 5                      |
| 数据存储   | CSV 文件（无数据库）            |
| 图表渲染   | PyEcharts（后端生成 iframe）    |
| AI 分析    | 讯飞星火大模型 API              |
| 机器学习   | scikit-learn + PyTorch (LSTM) |

数据文件路径约定（相对于项目根目录）：

| 文件名                  | 用途                   |
|------------------------|------------------------|
| `冰柠销量数据.csv`       | 月度样本源数据          |
| `模型评估结果.csv`       | 各模型训练评估指标      |
| `未来12个月预测结果.csv` | 各模型未来销量预测值    |

## 3. 数据中心

### 3.1 概览信息

页面顶部展示四项关键摘要：

- **样本条数**：当前月度样本总行数
- **起始月份 / 结束月份**：样本覆盖的时间范围
- **模型数量**：当前已训练完成并产出预测结果的模型数

### 3.2 月度样本维护

支持对历史月度销量样本进行新增、编辑和删除操作。

**表单字段说明：**

| 字段       | 格式说明                        |
|------------|-------------------------------|
| 月份       | `YYYY-MM` 格式，如 `2025-06`  |
| 销量       | 数值，单位：万支                |
| 上月销量   | 数值，单位：万支                |
| 去年同月销量 | 数值，单位：万支，可留空       |
| 节假日     | 0（否）/ 1（是）               |
| 促销       | 0（否）/ 1（是）               |
| 地区       | 文本，支持已有值联想            |
| 产品系列   | 文本，支持已有值联想            |

**操作按钮说明：**

- **新增数据 / 保存修改**：提交当前表单，自动区分新增与编辑状态
- **清空表单**：将表单切换回"新增"状态，清空所有字段
- **重新运行预测脚本**：异步启动离线训练任务，轮询完成状态后刷新预测结果
- **导入 Excel**：选择 `.xlsx` 或 `.xls` 文件后点"开始导入"，批量写入样本数据

### 3.3 Excel 批量导入规则

- Excel 文件支持**中文列名**（与界面一致）或**英文字段名**（`date`、`sales` 等）
- 必须包含所有字段，不允许缺列
- **遇到已存在月份时自动跳过，不覆盖、不报错**，导入完成后展示跳过统计
- 导入成功后样本表自动刷新

### 3.4 全部样本表格

展示所有月度样本，表头使用中文字段名（含单位"万支"），点击"编辑"可将该行数据带入上方表单进行修改，点击"删除"可移除该条记录。

## 4. 模型评估

展示训练脚本产出的各模型评估指标，包括：

| 指标        | 说明                                                 |
|-------------|------------------------------------------------------|
| 模型        | 模型名称（随机森林、梯度提升、极端随机树、线性回归、ARIMA、LSTM） |
| MAE         | 平均绝对误差（越低越好）                              |
| RMSE        | 均方根误差（越低越好）                               |
| MAPE(%)     | 平均绝对百分比误差（越低越好）                        |
| 置信度(%)   | 预测值落在真实值 ±10% 容差带内的比例（越高越好）      |

> **置信度说明**：置信度由训练脚本在测试集上实际计算得出，反映模型预测稳定性，非 MAPE 推导占位值。

---

## 5. 预测结果展示

### 5.1 查询与筛选

- **月份检索**：输入月份关键字（如 `2026`）可过滤显示匹配行
- **重点模型**：选择某个模型后，右侧"模型详情"区展示该模型的评估指标和预测预览

### 5.2 预测表格

- 数值统一保留 **2 位小数**并追加单位 **万支**
- 月份列保持原样

### 5.3 星火 AI 分析

点击"星火 AI 分析"后，系统将当前**筛选月份范围**内的预测数据发送给讯飞星火大模型，从整体趋势、季节性波动、经营建议三个维度生成简明分析文本（300 字以内）。

> **注意**：AI 分析基于当前月份筛选结果，若未设置筛选条件则分析全部预测月份；若筛选后有多个月份，则作为整段时间趋势进行分析。

---

## 6. 图表中心

### 6.1 筛选控件

- **模型**：选择用于系列预测图和单模型预测图的模型
- **系列**：选择"全部系列"展示各系列对比折线；选择某一系列则聚焦该系列展示

### 6.2 图表列表

| 图表名称               | 说明                                                   |
|------------------------|--------------------------------------------------------|
| 历史销量趋势图          | 历史月度销量折线图                                     |
| 多模型未来预测图        | 所有模型的未来 12 个月预测折线对比图                  |
| `{模型名}` 预测图      | 所选模型的单独预测曲线                                 |
| 各系列未来销量预测图    | 基于历史系列占比拆分的各系列未来销量图（可按系列筛选） |

> **系列预测说明**：系列未来销量基于历史总销量中各系列的占比，与所选模型的总体预测值相乘得出，属于可视化拆解结果，非独立分系列训练的预测模型。

## 7. 输入数据格式要求

训练脚本与 Excel 导入均依赖统一的月度样本格式：

| 字段名（英文）          | 含义           | 类型    | 是否必填 |
|------------------------|----------------|---------|----------|
| `date`                 | 月份           | YYYY-MM | 是       |
| `sales`                | 销量（万支）   | 数值    | 是       |
| `last_month_sales`     | 上月销量       | 数值    | 是       |
| `last_year_same_month` | 去年同月销量   | 数值    | 否       |
| `is_holiday`           | 是否节假日     | 0 / 1   | 是       |
| `is_promo`             | 是否促销       | 0 / 1   | 是       |
| `region`               | 地区           | 文本    | 是       |
| `product_series`       | 产品系列       | 文本    | 是       |

Excel 文件也可使用界面显示的**中文列名**，系统会自动识别并转换。

## 8. 常见操作流程

### 新增月度样本并重新训练

1. 进入**数据中心**，在"新增月度样本"表单中填写月份和各字段
2. 点击"新增数据"保存
3. 确认"全部样本"表格显示新增行
4. 点击"重新运行预测脚本"，等待任务完成提示
5. 进入**预测结果**页查看更新后的预测值

### 批量导入 Excel

1. 准备符合格式要求的 Excel 文件（中文或英文列名均可）
2. 在数据中心点击"导入 Excel"选择文件
3. 点击"开始导入"
4. 查看导入结果提示（总行数、成功数、跳过月份）
5. 如需更新预测，点击"重新运行预测脚本"

### 查看某月份 AI 分析

1. 进入**预测结果**页
2. 在"月份检索"中输入目标月份（如 `2026-03`）
3. 选择"重点模型"
4. 点击"星火 AI 分析"，等待结果展示

## 9. 依赖与环境

**后端依赖（requirements.txt）：**

```
Django>=5,<6
openpyxl>=3.1
```

> `openpyxl` 用于 Excel `.xlsx` 文件解析，导入功能必需。

**训练脚本依赖（另行安装）：**

- `scikit-learn`
- `torch`（PyTorch，用于 LSTM 模型）
- `pandas`、`numpy`、`matplotlib`、`statsmodels`

**前端：**

```
node + npm（或 pnpm）
npm install
npm run dev   # 开发模式
npm run build # 生产构建
```
